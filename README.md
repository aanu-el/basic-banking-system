# Basic Banking System Web Application

### Features:
- Dummy data in mobile
(sqlite)
database
for
upto
10
users.
User
table
will
have
basic
fields
such
as
name,
email,
current
balance
etc.
Transfers
table
will
record
all
transfers
happened.

- Flow:
Home
Screen
>
View
all
Customers>
Select
and
View
one
Customer>
Transfer
Money>
Select
customer
to
transfer
to
>
View
all
Customers.

- No
Login
Page.
No
Customer
Creation.
Only
transfer
of
money
between
multiple
customers
.

### Built with:
- HTML/CSS
- PHP
- MySql
